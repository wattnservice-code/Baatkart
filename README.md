# Batkart

Nettbrett-app for båtnavigasjon med kart, dybdeinformasjon, fiskeplassnavn og fartsvisning.

## Funksjoner
- OpenStreetMap + OpenSeaMap nautisk overlay (dybdesonder, bøyer, grunner)
- GPS båtposisjon i sanntid
- Kompass-retning på båtikon
- Sporlogging (start/stopp/slett)
- Fiskeplass-markører med navn
- Fartsvisning i knop

## Tech Stack
React, TypeScript, Vite, Leaflet.js, OpenSeaMap, Zustand, Tailwind CSS

## Kjør lokalt
```
cd webapp
npm install
npm run dev
```
